Projeto desenvolvido na trilha rápida de Desenvolvimento de Software do SCTEC, iniciativa do Governo de Santa Catarina em parceria com o SENAI/SC. A proposta deste repositório é praticar fundamentos de Frontend com uma página institucional simples em HTML.

## Sobre o SCTEC

O SCTEC faz parte de um programa de capacitação tecnológica voltado à formação prática em áreas como tecnologia e desenvolvimento de software. Nesta trilha, o foco é experimentar a construção de interfaces, organização de conteúdo e apresentação de um perfil profissional na web.

## Sobre o projeto

Este projeto é uma página estática com:

- apresentação pessoal;
- lista de habilidades;
- área de projetos;
- links para GitHub e LinkedIn.

## Tecnologias utilizadas

- HTML5

## Como executar

1. Abra o arquivo `index.html` em um navegador.
2. Se preferir, use a extensão Live Preview ou um servidor local simples.

## Estrutura

- `index.html` - página principal do projeto.
- `.vscode/settings.json` - configuração do editor para facilitar o fluxo de trabalho.

## Objetivo

Registrar a evolução inicial do aprendizado em Frontend dentro da trilha rápida do SCTEC, com uma base simples para futuras melhorias de layout, semântica e responsividade.
